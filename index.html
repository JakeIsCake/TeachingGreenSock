Getting started with ScrollMagic

What is it?

ScrollMagic is a javascript library that allows you to easily animate and pin elements, add callbacks, and Toggle between CSS classes of elements based on a user’s scroll position. It also allows you to easily add parallax effects and create an infinitely scrolling page.

Learning From Examples On This Site

Example 1: Fading Elements based on Scroll-Position

This can be accomplished by creating two versions of the CSS and toggling between them in the Javascript. ScrollMagic adds a controller, a scene, trigger elements, and toggles. You can add a transition effect in the CSS to make the change more fluid.

var controllerName = new ScrollMagic.Controller();
var sceneName = new ScrollMagic.Scene({
- triggerElement: ‘.css-name1’
});
.setClassToggle (‘.css-name1’, ‘css-name2’);

Example 2: Layered Parallax Transitions

This can be done by first building a tween, and adding layers then assigning it a position.

$(function () {
- Var tween = new TimelineMax ()
- - add([
- - Tweenmax.to(“#parallaxContainer .layer1”, 1, {backgroundPosition: “-40% 0”, ease Linear.easeNone}),
- - Tweenmax.to(“#parallaxContainer .layer2”, 1, {backgroundPosition: “-500% 0”, ease Linear.easeNone}),
- - Tweenmax.to(“#parallaxContainer .layer3”, 1, {backgroundPosition: “-225% 0”, ease Linear.easeNone}),
]);

After adding layers and assigning positions, a scene must be built where tweens and pins must be set, and indicators and controllers can be added. Each Scene has a duration and an offset.

var scene = new ScrollMagic.Scene({triggerElement: “#parallaxContainer”, duration: 2000, offset: 420})

- .setPin(“#ParallaxContainer”)
